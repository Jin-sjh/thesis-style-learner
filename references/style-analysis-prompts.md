# 风格分析详细提示词模板

当执行各维度分析时，使用以下内部提示词指导自身分析。

---

## 句式结构分析提示词

```
分析以下文本的句式特征：
1. 随机抽取 20 个句子，统计：
   - 平均字数（中文）/词数（英文）
   - 主动句占比（含"本文"、"我们"为主语）
   - 被动句占比（含"被"、"由"，或英文 is/are/was/were + 过去分词）
2. 统计以下连接词出现频率：
   - 序列类：首先、其次、然后、最后、第一、第二
   - 转折类：然而、但是、尽管、虽然
   - 因果类：因此、所以、由于、从而
   - 并列类：此外、同时、另外、不仅…而且
3. 统计括号内补充说明的平均每页频率
4. 输出：最典型的 3 种句式模板（去除内容，保留结构）
```

## 段落逻辑分析提示词

```
分析以下文本的段落结构：
1. 取连续 10 个完整段落
2. 对每个段落标注：
   [TS] 主题句位置（首/尾/无）
   [SP] 支撑方式（举例/数据/引用/推理）
   [LEN] 段落句数
3. 总结最常见的段落展开模式（如：主题句→现象描述→原因分析→解决思路）
4. 记录段落间的过渡句模式
```

## 学术语气分析提示词

```
从文本中提取以下语气标志词：
1. 自我指称方式：列举所有第一人称用法
2. 强调短语（每隔5页提取一次）：
   - 中文：值得注意/特别地/不同于/有别于
   - 英文：notably/importantly/in contrast/unlike
3. 引出结论的固定搭配：
   - 中文：实验结果表明/通过实验验证/由此可见
   - 英文：experiments show/results demonstrate/as shown in
4. 对比他方方法时的措辞（直接 vs 委婉）
5. 输出：10个最具代表性的学术套语（句型框架，不含内容）
```

## 章节开头/结尾范式提取

```
对每个主要章节，提取：
1. 开头2-3句话的结构模式
2. 章节结尾（小结）的结构模式
3. 子节之间的过渡句模式
输出为可复用的框架模板，用[...]标注内容占位符
```
